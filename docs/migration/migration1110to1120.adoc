=== coffee-module-redisstream

* Bevezetésre kerül a konfigurációban a pool-ra hivatkozás. Ezáltal konfigurációban megadható, hogy
adott redis melyik pooljából dolgozzon.

==== Átállás
* `a yaml konfigurációt szükséges a lentiek szerint igazítani.

régi:
```
coffee:
    redisstream:
        sampleGroup:
            stream:
                maxlen: 10
                read:
                    timeoutmillis: 60000
            consumer:
                threadsCount: 2
                retryCount: 2
            producer:
                maxlen: 10000
                ttl: 300000

```
új:
```
coffee:
    redisstream:
        sampleGroup:
            stream:
                read:
                    timeoutmillis: 60000
                connection:
                    key: auth
            producer:
                maxlen: 10000
                ttl: 300000
                pool: custom1
            consumer:
                threadsCount: 2
                retryCount: 2
                pool: custom2

```
