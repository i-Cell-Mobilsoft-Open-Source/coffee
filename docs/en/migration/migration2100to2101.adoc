==== Migration

Replace imports from:
[source,java]
----
import redis.clients.jedis.Jedis;
----
to:
[source,java]
----
import redis.clients.jedis.UnifiedJedis;
----

In the `RedisManager` calls replace the `Jedis::` to `UnifiedJedis::`

If you use `RedisManager` with pipeline, then the returned class changed from `Pipeline` to `AbstractPipeline`.
Old:
[source,java]
----
Pipeline pipeline = redisManager.run(Jedis::pipelined, "pipelined move").orElseThrow();
----

New:
[source,java]
----
AbstractPipeline pipeline = redisManager.run(UnifiedJedis::pipelined, "pipelined move").orElseThrow();
----
